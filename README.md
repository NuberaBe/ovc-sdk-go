# open-v-cloud

Golang client for [OpenvCloud (OVC)](https://docs.gig.tech).  
Mainly intended for the [OVC Terraform provider](https://github.com/gig-tech/terraform-provider-ovc).

# Installation

```bash
go get -u github.com/gig-tech/ovc-sdk-go
```
